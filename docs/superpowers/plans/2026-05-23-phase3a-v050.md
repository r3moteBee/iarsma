# Phase 3a Implementation Plan (v0.5.0)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire MCP write tools (mail.send, mail.modify, mail.delete), per-agent identity (issuance + revocation), scope enforcement, and the policy seam through the MCP server — so external agents can act on mail, gated by scoped tokens and dry-run previews.

**Architecture:** Extends the existing MCP handler pattern (session-get, mail-draft) with three new write handlers. Adds `TOOL_SCOPES` to the MCP dispatcher for call-time and list-time scope filtering. Introduces the `AgentTokenIssuer` interface (pluggable, Stalwart-OIDC default) and a `PolicyEngine` seam (v1: allow-all). Agent Settings UI for token management. Each new capability follows the contract → jmap-client → invoker → provenance pipeline established in Phase 2.

**Tech Stack:** TypeScript, Vitest, React, Jotai, JMAP (RFC 8620/8621), OIDC (RFC 7009/7662), IndexedDB

**Spec:** `docs/superpowers/specs/2026-05-23-phase3-design.md` — Cut 3a (items 1, 2, 3, 4, 13)

**Plans for Cuts 3b (v0.6.0) and 3c (v0.7.0)** will be written after Cut 3a ships.

```typescript
// shell/src/runtime/policy-engine.ts
import type { DryRunPreview } from './types.js';

export type PolicyEvaluationContext = {
  readonly toolName: string;
  readonly callerIdentity: {
    readonly id: string;
    readonly name?: string;
    readonly scopes: readonly string[];
  };
  readonly dryRunPreview: DryRunPreview<unknown>;
};

export type PolicyDecisionResult =
  | { decision: 'allow' }
  | { decision: 'deny'; reason: string }
  | { decision: 'require_approval'; reason: string };

export interface PolicyEngine {
  evaluate(ctx: PolicyEvaluationContext): Promise<PolicyDecisionResult>;
}

export function allowAllPolicyEngine(): PolicyEngine {
  return {
    evaluate: async () => ({ decision: 'allow' as const }),
  };
}
```

```typescript
// shell/src/runtime/agent-metadata-store.ts
export type AgentMetadata = {
  readonly tokenId: string;
  readonly name: string;
  readonly scopes: readonly string[];
  readonly issuedAt: string;
  readonly expiresAt: string;
  readonly revoked: boolean;
  readonly issuanceLogEntryHash: string;
};

export interface AgentMetadataStore {
  save(record: AgentMetadata): Promise<void>;
  get(tokenId: string): Promise<AgentMetadata | null>;
  listAll(): Promise<readonly AgentMetadata[]>;
  markRevoked(tokenId: string): Promise<void>;
}

export function inMemoryAgentMetadataStore(): AgentMetadataStore {
  const records = new Map<string, AgentMetadata>();
  return {
    save: async (r) => { records.set(r.tokenId, r); },
    get: async (id) => records.get(id) ?? null,
    listAll: async () => [...records.values()],
    markRevoked: async (id) => {
      const r = records.get(id);
      if (r !== undefined) records.set(id, { ...r, revoked: true });
    },
  };
}
```
